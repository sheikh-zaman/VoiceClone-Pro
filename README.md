## üöÄ Setup Instructions

### Prerequisites

No special prerequisites required. The project uses CDN-hosted libraries, so an internet connection is needed for the first load.

### Installation Steps

1. **Clone or Download** the project files to your local machine

2. **Open the Project**
   - Simply open `index.html` in any modern web browser
   - Or use a local development server for better performance:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     
     # Using PHP
     php -S localhost:8000
     ```

3. **Access the Website**
   - If opened directly: `file:///path/to/index.html`
   - If using a server: `http://localhost:8000`

### Browser Compatibility

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
